Viewing Logged Events for any Site

How to view the event logs for a single site or for all the sites within this DNN installation using the Log Viewer module.

  1. Navigate to Admin > Event Viewer - OR - Navigate to a Log Viewer module.
  2. Expand the Viewer section.
  3. At Website, select one of the following options:
    • Select All to view events from all sites. This is the default option.
    • Select the required site name
  4. Optional. At Type, select an event type to filter events by that event type.
  5. Optional. Click on an event record to view full details of that event. Re-click the event record to close it.
