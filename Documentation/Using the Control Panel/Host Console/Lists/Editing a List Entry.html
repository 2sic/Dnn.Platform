Editing a List Entry

How to edit a list entry within a list maintained on the Host Lists page. E.g. Victoria in the Australian Countries List

  1. Navigate to Host > Lists.
  2. In the Lists section, navigate to and select the title of the required list. This displays the details of this list.
  3. Click the Edit button beside the list entry to be edited.


  1. Edit the entry details as required.
  2. Optional. Click either the Move Entry Up or Move Entry Down button repeatedly to change the position of the entry in the list.
  3. Click the Save link.
