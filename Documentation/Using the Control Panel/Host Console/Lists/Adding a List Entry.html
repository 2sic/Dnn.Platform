Adding a List Entry

How to add a new list entry to a list using the Host Lists page.

  1. Navigate to Host > Lists.
  2. In the Lists section, navigate to and select the title of the List where the new entry will be added. E.g. Documents. This displays details of this list to the right.
  3. Click the Add Entry link. This displays fields enabling you to enter the details for the new entry.

  1. In the Entry Text text box, enter the data for the entry E.g. Manuals
  2. In the Entry Value text box, enter the identifier for the entry E.g. Manuals

  1. Click the Save link. The newly added entry is now displayed.

  1. Optional. Repeat Steps 3-6 to add additional entries to the list.
