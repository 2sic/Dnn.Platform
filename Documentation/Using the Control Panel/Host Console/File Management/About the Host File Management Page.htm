About the Host File Management Page

The Digital Asset Management module that is located on the Host > File Management page allows SuperUsers to access and manage to all files across all sites within this DNN installation. This administration module can be added to any site page.

For details on using this module, See "About the Digital Asset Management Module"

Installation Note: This module is installed on your site.

Module Version: The version number is always the same as the DNN framework version number.
