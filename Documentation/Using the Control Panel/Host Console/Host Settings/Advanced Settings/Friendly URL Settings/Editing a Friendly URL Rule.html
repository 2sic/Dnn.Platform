Editing a Friendly URL Rule

How to edit a friendly URL rule.

  1. Navigate to Host > Host Settings.
  2. Select the Advanced Settings tab.
  3. Expand the Friendly URL Settings section.
  4. Click the Edit Rule button beside the rule to be edited.

  1. In the Match text box, edit the details as required.
  2. In the Replace With text box, edit the details as required.
  3. Click the Update Rule button.

Tip: To cancel editing a rule, click the Cancel Edit button.
