Setting Authenticated Cacheability

How to set the Cache-Control HTTP header value for authenticated users.

  1. Navigate to Host > Host Settings.
  2. Select the Advanced Settings tab.
  3. Expand the Performance Settings section.
  4. At Authenticated Cacheability, select from these options:
    • NoCache
    • Private
    • Public
    • Server
    • ServerAndNoCache
    • ServerAndPrivate

  1. Click the Update button.
