Loading Additional Blog Files to the Web browser

How to enable the Blog module to load additional files to the client (typically your Web browser) when the blog is rendered. This allows you to choose a stylesheet or enable syntax highlighting.

  1. Select Module Options from the View_Blog module actions menu. This opens the Blog Module Options page.
  2. Go to the Load Additional Files section.
  3. At Add jQuery, to enable the jQuery library (http://www.jquery.com) - OR - to disable.
  4. At Files On Host, beside one or more of the listed files. supplied with the Blog module by default:
    • css/shCoreDefault.css: to select a new stylesheet.
    • js/syntaxHighlighter.js: to enable this syntax highlighting tool. Syntax highlighting displays any code entered into the body of a blog entry "as is", rather than displaying the text as formatted.
  5. At Files On Portal, beside one or more of the listed files. The listed files are managed under [site home directory]/Blog/Include.
