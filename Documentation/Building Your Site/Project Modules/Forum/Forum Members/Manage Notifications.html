Manage Notifications

How to manage your notifications for forums and threads which you have chosen to track using the Forum module.

  1. Select My Settings from the module actions menu - OR - Click the My Settings link. This opens the User Settings page. The Control Panel > Overview window is displayed.
  2. In the right-hand panel, select Control Panel > Notifications. This displays the Tracked Forums and Threads window with the Forums tab selected.
  3. On the Forums tab, you can perform the following:
    • Remove Tracking: Click on the Delete button beside a thread to stop tracking it.
    • Go to Forum: In the Forum column, click on the [Forum Name] link to go to that forum.
    • View Last Post: In the Last Post column, click on the [Date/Time] link to view the last post.
