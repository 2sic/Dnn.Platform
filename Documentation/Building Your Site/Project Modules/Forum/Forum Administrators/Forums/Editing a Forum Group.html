Editing a Forum Group

How to edit the name of a forum group.

  1. Go to the Forum Administration page. See "Navigating to the Forum Admin Page"
  2. In the left-hand panel, select Forums > Forum Manager. This displays the Forum and Group Management window.
  3. Click the Edit Group button beside the required forum group.

  1. Edit the forum group name in the exposed text box.
  2. Click the Update button.
