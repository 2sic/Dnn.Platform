Settings for Calculated Columns

How to set the data within a calculated column to be displayed on the Edit Form page of the Form and List module.

  1. Select  Form and List Configuration from the Form and List module actions menu - OR - Click the Form and List Configuration  button.
  2. Maximize  the Schema Definition section.
  3. Maximize the More Column Settings section.
  4. Go to the Settings for Calculated Columns field.
  5. At Render In Form, render on the Edit Form page - OR - to disable.
  6. Click the Save Configuration and Return link.
