Configuring Journal Settings

How to configure the settings applied to an instance of the Journal module.

      2. At Enable Photo Attachments, select from these options:

  4. At Default Page Size, select the number of journal items that are displayed per page. The default setting is 20.

  6. At Journal Types, select the types of journals you want to display.
